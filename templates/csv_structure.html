CSV file structure

CSV file format

We need a csv file to describe the sequencing files you will upload.

"CSV" stands for "Comma Seperated Values" and you can read more about it at the relevant wikipedia article

You can open it on a text editor, or in a spreadsheet application (for example MS excel, OpenOffice Sheets, Google Sheets), as long as you save it at the end as ".csv"

Preparation of your data.

  1. Download and use the sample csv file as template.
  2. Fill in the csv for the files you want to upload. The column names are explained bellow.
  3. Please note that one line could refer to multiple of your files

Column names for the csv file

  1. Sample_ID: The number or name assigned to your sample, can be a number or a name used to indicate the identity of the sample. E.g Kazakhstan_1, Control_1, K1, 1.
  2. Sequencer_ID: The name of the sequencing file output by the sequencer. This may be arbritrary (i.e 001id001) or may be the same as the Sample_ID. This must be the exact ID excluding the shared file ending. For example, if your file outputs are named 001id001_S65_R1_001.fastq.gz, only include up to the first underscore, i.e 001id001.
  3. Sequencing_provider: Company, organization or individual that sequenced samples.
  4. Project: You will be given a project name, this will be the same for all samples within a project. I.e sl-kazakhstan23
  5. Region: Amplicon region (SSU, LSU, ITS1, ITS2)
  6. Index_1: [Optional] The sequence of index 1 used for sequencing, i.e GACGAGATTA
  7. Index_2: [Optional] The sequence of index 2 used for sequencing, i.e ACATTATCCT.
