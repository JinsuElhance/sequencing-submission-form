CSV file structure

CSV file format

We need a csv file to describe the sequencing files you will upload. This must be filled in correctly to allow file upload and quality control analyses.

"CSV" stands for "Comma Seperated Values" and you can read more about it at the relevant wikipedia article

You can open it on a text editor, or in a spreadsheet application (for example MS excel, OpenOffice Sheets, Google Sheets), as long as you save it at the end as ".csv"

Preparation of your data.

  1. Download and use the sample csv file as template.
  2. Fill in the csv for the files you want to upload. The column names are explained bellow.
  3. Each row should refer to one pair of files, if using Illumina paired-end sequencing. Do not include separate rows for the R1 and R2 files.

Column names for the csv file

  1. Sample_ID: The number or name assigned to your sample that you can use to identify it. Please keep this as short as possible, preferably under 10 characters. Can be a number or a name used to indicate the identity of the sample. E.g Kazakhstan_1, Control_1, K1, 1.
  2. Sequencer_ID: The name of the sequencing file output by the sequencer, must match exactly the characters in the file names, (without the file ending).
    For Illumina sequencing, include the name of the pair of files up to the the '_S' characters.
    Example 1: For Illumina sequencing if your file pairs are called '055id055_S361_R1_001.fastq.gz' and '055id055_S361_R2_001.fastq.gz', write '055id055' only.
    Example 2: Non Illumina sequencing. If your file is named 'KlmtM2_M11.fastq.gz', please write 'KlmtM2_M11'
  3. Sequencing_provider: Company, organization or individual that sequenced samples.
  4. Project: You will be given a project name, this will be the same for all samples within a project. I.e sl-kazakhstan23
  5. Region: Amplicon region (SSU, LSU, ITS1, ITS2)
  6. Index_1: [Optional] The sequence of index 1 used for sequencing, i.e GACGAGATTA
  7. Index_2: [Optional] The sequence of index 2 used for sequencing, i.e ACATTATCCT.
